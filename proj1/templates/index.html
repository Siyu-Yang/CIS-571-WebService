A Simple Word Game



This application is created to help you learn English words!


I build this web application as my first project in CIS 571 course. It is an English words based game. It bases on 5000 most common words in English, try to guess all of them! There are four APIs used to achieve the goal:

  • Oxford Dictionaries API
  • Google Cloud Translation API
  • REST API: Register API, Login API
  • SOUP API: Get_wordlist API
